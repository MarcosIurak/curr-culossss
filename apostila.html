Tags HTML

 <html>, <head>, <body>,<h1>, <p>, <ul> <li>, etc.

  • A tag <html>, é o elemento raiz de um documento HTML. Ela envolve todo o conteúdo da página da web e define o início do documento HTML.
  • A tag <head> contém informações sobre o documento HTML que não são visíveis na página da web. Ela geralmente inclui metadados, como o título da página, links para folhas de estilo CSS, scripts JavaScript, informações de codificação e outros elementos que ajudam a estruturar e descrever a página.
  • A tag <body> contém o conteúdo visível da página da web, como texto, imagens, links e outros elementos que são renderizados no navegador. É dentro da tag que a maior parte do conteúdo de uma página da web é colocada.
  • A tag <h1> é usada para criar um cabeçalho de nível 1. Ela representa o título principal da página ou a seção mais importante do conteúdo. Normalmente, as páginas têm apenas um <h1>, que é seguido por cabeçalhos de nível inferior, como <h2>, <h3>, etc., para organizar o conteúdo hierarquicamente.
  • A tag <p> é usada para criar parágrafos de texto. Ela envolve blocos de texto e cria espaçamento adequado entre os parágrafos. O conteúdo dentro da tag <p> é tratado como um parágrafo separado.
  • A tag <ul> é usada para criar listas não ordenadas, enquanto a tag <li> é usada para criar itens de lista dentro de uma lista não ordenada. Itens de lista em uma lista não ordenada são geralmente representados com marcadores, como pontos ou círculos.

CSS

Nesta seção, abordaremos conceitos básicos de CSS.

  • Para mudar as cores de fundo (background) e de texto (letras) em CSS, você pode usar as propriedades 'background-color' e 'color', respectivamente.
  • A propriedade 'color' é usada para definir a cor do texto dentro de um elemento HTML. Da mesma forma, você pode especificar a cor usando nomes de cores, códigos de cores hexadecimais ou valores RGB.

Tamanho das letras

Você pode alterar o tamanho da fonte em CSS usando várias unidades de medida mas epresentarei apenas a (%). Cada unidade tem suas próprias características e aplicações específicas.

  • elemento { font-size: 150%; /* 150% do tamanho da fonte do elemento pai */ }
